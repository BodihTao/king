皇权的能量场
作者     幻狮王（旺秋札巴藏巴拉）著
于2015年4月11日开示


        今天的事，终算过去了；随着两位退群（原QQ群，因种种原因，现已解散）的网友，无诤地自动退群，我也不用再担心今天的麻烦了，阿弥陀佛！
        我昨晚打坐已知，今天终算平静，阿弥陀佛！
        我现在终于可以放心地讲两句闲话了。
        “金胡子”菩萨呀，您知道我为什么让管理员通知您五月份不用来面授了吧？您本来早已寄来学费，准备五一假期来学法；而我却不让您来，我已传给您了。
        换着别人，早就生邪见了：“什么意思？收了学费，却不教了？”
        “金胡子”是什么人？会如此无智？那他就不是“金胡子”了；而是“花白杂胡子”了。
        我当时令“红胡子”，更名为“金胡子”，就在观他的缘起宿命何在。我只是提醒他，让他将缘起法名“金胡子”，再配一幅头像图——要么是斯大林，要么是欧洲其他皇权者。结果他找到“斯大林”，及“查理大帝”两幅图。我一见“查理大帝”，便知是他的天命所归。
        我当下便用宗教手段（即无上密部特殊加持法），将常存于宇宙间的“查理大帝”心识能量，以密咒聚集，并调动起来，刹那间射入“金胡子”的身心灵……并以梵文“嗡阿吽”种子字，种于“红胡子”顶喉心，再以安神“钉橛”、永住密咒及具誓祈愿等，为之封印……
        我讲了一句全世界佛教徒都不敢讲的话——“查理大帝”是“金胡子”的本尊。
        众所周知，只有佛菩萨才称本尊；一个拿十字架权杖的外道帝王，岂可称为本尊？但我自有密意；不能擅自泄密，抱歉！
        一切外在的形式，都是无常、无我的空性，没什么可执著的；只是找到适合自己的宇宙能量（能量守恒这个现代词话，不知能否说明那种心识能量信息），一气贯通，原本如是、本自圆满。
        任何佛菩萨的种子字、法器、手印等，都有密意；只看您是否有缘有福得到传承而已。但绝不是目前市面上解释的那样；密宗念诵法本中的解释，全是浅表义，密意是口耳相传的。
        仅就“查理大帝”这个世俗君王，其手中的权杖，及有求必应的金珠，就有世俗的象征意义，及微量物质能量信息在空间波动。他头戴皇冠，象征上天的贵气；右手握权杖，象征至高无上的皇权；左手平端有求必应的金球，象征无量无边财富的福气；双眼上视，象征“恩威并用”及“上天使者”的霸气。
        具此缘起物的象征意义，及其缘起物所散发出的物质能量信息——贵气、福气、霸气、皇权等，就是王！——此乃修心层次的虚拟世界；必须在现实世界广结善缘、各种包装，及努力奋斗，便可成为行业的皎皎者。
        大家看，你们家的龙王是否也如此？只是权杖与金球，换了一下左右手而已。
        按照中国习俗——左大右小，龙王所持器物的方向，更适合中国人。
        若无此虚拟世界的内心修持；仅凭现实世界的各种奋斗，很难如愿，除非命中（业力因果）本有。
        当然啦！仅凭内心修心，而不到现实生活中，去广结善缘及奋斗，也不可能如愿以偿的。如此内外结合、一气呵成，方可改变命运！
        金胡子”，您“恩威并用”强大意志（绝对掌控），就是霸气。
        您那气定神闲、悠然自得的动作缓慢柔顺稳健、贵者语迟语缓、爱语柔和甜美、心宁心静等，就是贵气。
        您心量广大、上供下施、功德无量、随喜快乐等，就是福（或富）气。
        您那一举手、一投足的权威气势动作（多研究毛泽东及斯大林的动作），就是改天换地的熏天皇权。
        我不可以泄密佛菩萨的手印、法器等密意；但可以泄露一丝帝王皇权（足够自信、主宰生死）的动作——讲话时，应以极少的动作，竖起一阳指（右手食指指天），代表上天的威严及天令，祝福或讲好话时指着对心窝，就是下令或赐福；批评及怒斥对方时，右手食指指地，对方将受罚……
        您真地“具足足够自信”、“绝对掌控及主宰对方生死”、“与查理大帝同一心识”，您将圆满具足他的霸气、贵气、富气、皇权（当然不可能当君主，也许是单位上老大，或单位里皎皎者等）。
        ——这只是内在修心，只是成功的一半；必须同时配合外在的现实中奋斗，才行。上次，某人逼我讲出我执与自信的区别；我岂可受人威胁？他还威胁我不讲，他就立刻退群（原QQ群，因种种原因，现已解散）；哈哈哈，唯有这个QQ名叫“山顶洞”的威协者退出，“皇权”与“山顶洞人”，原本就不在一个层次上吗！我才能讲出今天的秘密呀！
        哪有什么“我执我慢”？全是佛门中的“佛执佛慢”，及世间帝王术中“王执王慢”呀！点到为止，留下福、寿、康、宁的广大空间，让大家共舞。晚安！
        多美多爽！
        尤其是皇冠顶上的尖顶，是与天相通的天线、秘密通道，亦相当于金字塔效应。我知道用这些现代词，做比喻不恰当；但那个尖顶，确实有密意。
        我不敢过于泄露天机，聪明人自己去观察，及体悟吧，大家会大吃一惊地！
        看看特殊古人的帽子顶上，及诸佛菩萨、神仙等头顶上发髻等，真地那么简单吗？还有建筑、避雷针等，真地毫无密意吗？

        以上是昨晚随缘闲聊了几句有关网友“金胡子”的QQ头像，仅仅是我个人的一己之见；不一定正确及有用。
        仅供参考！
        不管是过去的老朋友，还是新朋友，愿大家永远都是知心的朋友。
        大家在这个群中，共同探讨改变命运、趋吉避凶、健康长寿、功成名就、发财致富、神通法力、解脱自在等共同话题。
        因为本群面向大众，我虽信佛；但并非只有三亿多佛徒一种声音，而其他六十多亿人口就没有话语权。
        因此，希望以后不要一而再、再而三地出现以佛教观点，来非礼本群。
        谢了！
        好了！各位菩萨，关于昨晚“金胡子”的随缘唠叨，应该画上句号了。
        只是“金胡子”本人，及各位有缘人，最好引入自相续，方得真实受用；否则，只是又听了一个故事而已，随着两晚上睡眠，全忘光了。
        大家看看生活中，凡是低头哈腰快步行走的人、讲话声音大语速快的人等，都是终生贫困者。
        还有喜欢摇头晃脑、坐着还抖脚晃膝盖的人等，都是终生贫穷人。
        哪有什么“贵相”，及“富相”之状？
        因此，仅从最低层面看，大家若能从昨晚“金胡子”的废话中，仔细揣摩“贵气”、“富气”、“霸气”、“皇权”等王者气质处，悟出真知灼见，以改变自己及家人的人生；总比大家目前强点吧！
        “金胡子”与有兴趣者，可通过录像、或其他渠道，仔细观看及揣摩毛泽东及斯大林的举手投足、一言一行等，他们绝对是顶级的帝王气质。
        如果能找出、并真实感受出“气定神闲、泰然自若”的内心素质，就能一改目前的急于赶路、急于辩解、急于挣钱的陋习。
        一个全国都是自己的帝王君主，集皇权、财富等于一身的帝王，他没必要向别人急于证明什么、解释什么，炫耀什么；他自然而然地处于“养尊处优、贵人语迟、指点江山”的涵养中。
        大家若能从模仿（演员表演皇帝，演的再好、也不是皇帝），进入实证这种贵气，而内涵丰富，自然吸纳无数追求者众星捧月。
        因为，“贵气”如磁铁，自然吸纳铁器，不会吸纳泥巴、木材，不须任何刻意及造作。
        当然啦，这里面的学问多着呢，我没时间及精力，一一写出；留待以后，与有缘人面授吧！
        以极度地、绝对地“佛执佛慢”，彻底根除“我执我慢”，就是无上密的究竟修法及现证法。就是“足够自信”！
        极度地、绝对的“王执王慢”，就是改运之王——将自己与生俱来的“欲望欲求”，及“匮乏感”等，彻底根除；代之以“安富尊荣”，及“绝对掌控主宰万民生死”的帝王旺运，成为世间大富大贵者。
        “金胡子”潇洒走一回，人生如意尽欢颜、日进斗金夜来香！
        来！朋友们，咱们共同祝贺“金胡子”——
举杯邀查理，
对面众嫔妃；
人生得意须进酒，
日进斗金夜来香。
