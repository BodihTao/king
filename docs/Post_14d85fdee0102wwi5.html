末法时代的现状

——幻狮王

于2015.02.27开示

 

        萨迦法王今年的新春贺辞中，公开点明开运纳福的密意:仅靠祈请、祈祷（三宝，及一切神灵），是没用的，必须自己首先成办缘起物。

        请看这个网上卖的金发丝水晶球，若放在任何财神面前修法，有此自动吸收宇宙能量、聚气聚财的天然水晶球缘起；肯定比什么都没有、或放人工造的假水晶，效果不一样！

        条件好的网友，请一个这样金发丝极顺，而毫无杂纹杂质的水晶球，供在财神前，再以宇宙空间之力，慑召天地能量及财气，肯定对个人财富产生助力！

        网上标价十八万元；更纯的金发丝水晶球，八十多万元。

        在网上查；但小心被骗，假的太多！

        我过去在各大居士家里，见的所有水晶球，都是假的。我的老家小河镇，就有一个专门造水晶球的小厂，成品与真的一样，全是特殊玻璃染色的。

        某网友菩萨，我很同情您的遭遇，您病地异常严重，几乎是残废人一样；但您最好心理阳光一点，我知道您命运的不公，但没必要总与环境较劲！

        您这阴暗的心理，会直接感召更严重的疾病及贫困！

        您也不必用鸟吃肉不吃素来嘲讽大家，您会更加贫困；但网友们不想像您一样残废及贫困！不要给网友们造成贫困的缘起！

        您只在南华寺接触过显宗；您不懂密宗的缘起法，是否要萨迦法王向您下跪、给您磕头、向您求法？若您答应，我可以通过萨迦寺大堪布向法王转达！

        有光明，就有黑暗，很正常！大家正要因金丝水晶球改运，却又冒出个贫病交加、愤世嫉俗的又病又穷的某先生，大煞风景，再次阻拦各大菩萨开运纳福的吉祥富贵缘起！

        这就是末法时代的悲剧！算了！各随天命吧！

        这正应了“真正受用的密法，必须面授”！看看现状！真正的开运纳福，各随因缘吧！

        管理员曾多次强调，一个人不要一次性发多张图片，浪费空间。

        好啦！今天不讲了，大家已失去富贵缘起了；今后也尽量少讲，因总有心理不健康的人出来捣乱。

        自己的贫困，应想办法改变，而开运纳福，不是怨天尤人地仇视社会，这样会越来越糟！

        既然某先生，不适应本“正能量”群体；蹋出去吧！

        再见！

        可惜呀！这么殊胜的金丝水晶球改运开财门的缘起法，被一位仇视社会、仇视人类的残废人，给彻底破坏了！大家认命吧！

        通过今天的惨痛教训，大家应铁定相信我过去讲过的话——结缘，要与有福报的人结缘！

        事实上，某先生的严重疾病，及极度贫穷，又不是我们给他造成的，在我们群中一而再、再而三地发什么飚！他多次在本群中，阴阳怪气地发表“狂禅”言论，及虚无主义的顽空断见，管理员禁言他几次，仍不思悔改，继续顶牛抬扛。他的狂疾及贫穷，是他自己的因果，应消业积福，方为正道；而不是以自己的心理不平衡，与天下人为敌！

        又如另一位先生，去年年底，我正讲实修实证；他别有用心地转发某法师的言论:“一个道场，只有既讲经，又讲实修，才是正法道场。若只讲经，不讲实修，就是邪法道场；或不讲经，只讲实修，也是邪法道场。”这句绝对的断语，是对是错，我不加妄断；我要讲的是任何修行，都是讲次第的。如，刚开始修行，一无所知，教理不明、次第不清，自然以学习经典理论为准为主，唯有明理知方向，方可不堕入盲修瞎炼的魔坑。  正如西藏格鲁派的宗风家风，就是广大的闻思，待一步一步学修菩提道次第，而证得一念间让菩提道三主要道，在心中无碍生起，及相续后，才有资格参加大灌顶，而观诵密宗外密法。待密宗见真正生起后，就将一切经典理论，落实在一坛法中，圆融无碍地修持无上密本尊及坛城等，何时见理论与实修分家的断修法？

        一个从未得过这类系列学修证大法的半瓶水，总喜欢一知半解地指责别人！

        没有佛法的空性，您能改运改命吗？在过来人的指导下，哪个实修法，脱离了三法印、四圣谛、十二因缘，及三十七道品的经典内容？穷分别什么？

        曾有一位在寺院拿了一本宣传册子的人，叫我拜她为师。她说佛陀悟的不究竟，要改革佛教、重写佛经。

        一个连1+1，等于几都不知道的幼儿，指责核导弹科学家们，不懂他所知道的“小猫小狗小刺猬”幼儿歌。

        我目前活着的索嘎上师，对佛学院的学僧们，讲经典理论；对禅修院的资深喇嘛，传授实修实证的无上密法。

        这位闭关二十年的资深大喇嘛，难道是邪法师？其佛学院，与禅修院，都是邪法道场吗？

        要知道，他在佛学院，是不讲无上密修证法的；他在禅修院，也从不讲经的。难道天下佛学院，及禅修院都是邪法道场？哪一天，世界上的显宗寺院及佛学院的高僧大德，不在讲经？他们极少讲修炼，难道他们都是邪法道场？！哪个闭生死关的岩洞修行者，在天天讲经、学经？上述哪个道场，不是将佛门的“信、解、行、证”，融为一体，而随顺不同的对象、次第，及层次等，而作不同的教学及随机开示？！分别什么！

        某法师，讲的每句话，都是究竟的？那他预测的两次人类大灾难，为何一次都没应验？！

        我尊敬每一位僧众，及佛教每一个法门，大家从我过去的博文中，可以找到印证。

        我只是希望个别人，无知可以，但应广大的闻思修，而成为智者。
