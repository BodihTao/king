第五节  最危险的羌塘高原无人区（蓝衫客系列小说之喜马拉雅复仇记）

By: 幻狮王 原文发布于：2018-06-13 16:38:36

《蓝衫客之“喜马拉雅复仇记”》
第五节
最危险的羌塘高原无人区
2018年5月30日

蓝衫客租了一辆老解放牌汽车，带上许多压缩饼干、干牛肉、大小不一的藏刀、棉衣、登山队员专用的睡袋、登山协会的帐篷等，驰向平均海拔4000米以上的西藏羌塘高原无人区。

无人区内，到处都是置人以死地的沼泽湿地、埋人的沙漠流沙、随时塌陷的冰层、永久冻土，及高寒缺氧等，令藏族司机心生恐惧。

司机不停地提醒：“这一望无际的无人区，根本没有任何参照物来辨方向。我不能再往前走了；否则，我找不到回去的路而迷了路，必死！每年都有很多人死在这里，很少见到有人活着走出这无人区……”

蓝衫客善解人意地让司机停车，帮助自己把车上的所有物品全部都卸下来。

司机临走前，像看着死人似地，怜惜地看着蓝衫客，并再次善意强调：“一定要找到一个洞穴，住进去，凭一把长刀，就能守住洞口，再多的饿狼，或棕熊，也只能一次性进一只；千万不可住帐篷，狼群四面八方疯狂袭击，就活不成了。”

蓝衫客感恩地拍拍司机大哥的肩：“谢谢你，老哥！”

蓝衫客一直目送着司机大哥，从视线中消失。

蓝衫客孤零零地一人站在青藏高原1/4的60万平方公里的羌塘无人区，在这人类的生命禁区；不免想起自己的遭遇，不禁阵阵心酸心凉……

蓝衫客将支撑帐篷的钢管一节节拧紧，上面绑着自己的鲜红色内衣，将钢管深深地插入食物旁边的泥地里，做标记。

然后，带着宽厚的长刀，及几把短刀，到处去寻找巨型动物的洞穴。

幸亏蓝衫客力大细心，另外一根一节节拧紧的支撑帐篷的钢管，在前方探路：有好几次，看到地面是冻土，异常结实，结果用钢管一捅，下面竟是几米深的稀泥沼泽地。如果走上去，一下子就陷入稀泥中窒息而死，根本没人爬得起来……

有的地方是地面上的沙漠，但地面下居然是深不可测的松沙，人一旦走上去，立即全身陷进去，而看不见一根头发。

与沙土地颜色相似相近的野狼，埋伏在沙土地低凹处，人类一旦走近它们；它们立即分工合作，腾冲而起，撕咬人的喉咙、脸部等。

蓝衫客找了几个小时，根本就没有什么洞穴。

眼看天快黑了，蓝衫客连兔子的洞都没有找到；只得抓紧时间扎帐篷露营了。

是夜，中午与晚上的温差竟在50度左右……

真的是小便落地成冰渣呢……

未完待续

为什么很多人进入西藏无人区就再也没有回来
