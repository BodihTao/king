金沙江畔德格印经院  

（幻狮王小小说）

2018年2月10日



大家住在相对最好的宾馆——德格县宾馆（当时的条件极差），开心的共叙沿途风光，尤其是金沙江，大家在那里沿江玩了一整天。         

大家计划次日去拜访名扬天下的德格印经院。         

次日上午十点左右，德格印经院开门不久，蓝衫客一行叩拜该印经院。         

蓝衫客首先大供养了当家喇嘛。         

僧众们各就各位，开始了一天繁忙的工作，大家在当家喇嘛的带领下，参观了僧众们的印经流水线全过程。         

最后，当家师打开巨大的门锁，推开厚重的两扇大门，里面是大藏经（《甘珠尔》、《丹珠尔》）的全部印经板（黑板、红板）。整个巨大、高大的大殿中，堆满黑色，和红色的长条印经板，及其他各类图案的分门别类的印刷板等。         

蓝衫客突然意识到，佛、法、僧三宝中，法宝最上最无上等，立刻大供养，及大礼拜。         

蓝衫客拿出巨款，供养藏文大藏经。

任何人，在佛、法、僧三宝面前，越是表现的厚道诚实，就越会得到真实的受用。

跪在《大藏经》前的蓝衫客，暗自思考：一切佛菩萨、辟支佛、阿罗汉、解脱自在的大成就者、高僧大德等圣贤，皆是依此三藏十二部经论，为语密所依所获得的无漏悉地；一切六道众生，也必须依此大藏经，为语业所依方可解脱轮回之苦。         蓝衫客突感“此法宝”的无上功德利益，及解脱的唯一绿色通道，而顿感自己目前的一切，全部都是此法宝所带来的；而且，未来的一切，也只有此法宝，才是自己及众生的不二所依，及究竟护法。         

蓝衫客跪在一排排分类的经板壁柜前，泪流满面的合掌感恩！         

“哇哇！哇——”悲惨欲绝的凄凉哭声，惊吓了众人的目光，卓玛扑倒在地上，双手紧紧抱住油腻腻的红色经版，嚎啕大哭！

站在欲死欲绝、伤心透顶的卓玛旁边的“鬼精灵”拉珍，哀怨地钉了一眼蓝衫客，也是凄楚的泪洗脸……         

蓝衫客从“鬼精灵”拉珍的眼神中，读出了卓玛之所以如此伤心欲绝的根本原因……

依然合十跪着的蓝衫客，回头看向富二代扎西，眼圈通红的扎西，扭头避开了蓝衫客搜索的目光……         

当家喇嘛看着抱着经板痛哭不止的卓玛的身缘起，不加思索地应缘授记（俗称预测）：“身不离法，法不离身，久后必成金刚身的法身如来！”         

全场闻言大惊：文殊道场的当家师，绝对不会随便乱讲话，必有深意。大家看看当家师，又看看葡伏在地上的抱着经板的卓玛，隐隐感觉到她久后必有大成！         蓝衫客由此当家师的缘起观，联想到八地菩萨仁钦巴上师对卓玛的授记，及江达喇嘛法性身的空谷回声（劝说蓝衫客与卓玛结合，共证菩提），蓝衫客已经明确的了知卓玛未来的无上悉地……

附点评：

          笨人才耍心眼儿，聪明人只需厚道！

上次由方某、易某某、杨某某、陈某某、冯某、卢某某等人自行发起，及供养的“姜太公神王令”木雕工艺品时，由于木雕师傅的小疏忽，将神王的九节打神鞭，误雕成八节鞭。他们并没有大发雷霆；反而不停的赞美木雕师傅的手工雕刻艺术天下第一，方某本人因此私人拿出一千元，嘉奖木雕师。做人做事，不要因为别人一时的疏忽、一点儿小过失，就刁钻刻薄、咄咄逼人，这样是缺福少慧的表现。我们在任何时候、任何地方、任何事情上，最好表现得厚道一点，对自己对别人都是好事！

 在这个时代的成功者，最关键的基本功，应该是做人做事问心无愧，自己安心、别人放心，才可能赢得人脉财气运势，才有可能成为人生的赢家！
