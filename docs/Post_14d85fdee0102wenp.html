幻狮王法力群宗旨

——幻狮王
于2015.01.31开示
各位菩萨，大家早上好！
        我再重申一下，我们这个群，是一个普通居士群，是一个以年轻人、初级信佛者的小平台，只有一个宗旨:大家学习工作之余的消遣地方，只求轻松愉快、开心快乐、喝茶吹牛嗑瓜子的露天草场；绝不涉及任何政治、宗教、色情、暴力的小小茶社，望大家配合。
        我只是一个普通居士，不兼任政治、宗教等职务，我没有任何责任，及义务，回答任何严肃的宗教问题。
        我不代表任何宗教、门派或宗派，我只是一个无业游民、社会的最下层，我只能代表我自己，我只能保证我自己做到:
诸恶莫作，

众善奉行；

自净其意，

是诸佛教。

        若有严肃的宗教问题，敬请前去找高僧大德解决。
        若有教理、实修仪轨等相关问题，敬请前去寻找汉、藏大和尚，及修持极好的各大居士们，给您详细解答；我不是佛学大师，我更不是佛菩萨的化身，我没资格成为佛的代言人，而随意发挥。
        我反复讲了，我从未自称上师，我只是中学教师出身，至于大家如何称呼，那是大家的权力，我无权干涉，而且一切名相都是空性的。
        我给自己的定位:网络写手。一个写神话故事、写东方玄幻的初级写手——唐家三少可是网络钻石作家；我吃西红柿，及天蚕土豆等二、三十岁的小伙子们，可是网络白金作家。
        我没这些可畏的后生们功夫深，连末流的网络作家都不是，仅仅是未入流的初级写手而已，所讲所写都只不过是蹩脚的神话故事而已，大家权当看神话小说吧。
        中国民间有句俗话：“看了西游记，说话像放屁。”仅此而已！
        哈哈！我很喜欢“唐家三少”等现代小家伙们的作家世界，孙悟空在他们的笔下，连一根猴毛都算不上；热罗多吉札与大威德金刚，在他们的笔下，连自己都保护不了。
        真爽啊！修法者一拳能毁天，一脚能灭地，一挥手重造宇宙，重塑三界！
        一静坐，从凡夫到地仙；一闭关，从地仙到天仙；吃一个金仙大丹，变仙皇！
        我宁可活在这种虚幻中，也不愿回到现实卖心脏！
        听说，某地目前对我恨得牙痒痒，既然奈何不了我，就在全国范围内，对其汉地居士诽谤我！
        我目前，仅仅是对我有缘的居士们，讲点神话，聊聊天，说实话，我身为佛教徒，都对目前佛教状况，没一点归属感——这就是佛教的衰落……
        我退出他们的队伍，我关闭佛教博客，我仅仅在QQ群中聊聊天，又没一分钱收入，对我个人有什么好处？没名没利遭诽谤，真是吃饱了撑地难受。
        所以，有缘的网友们，请不要让我为难，我不回答任何问题，我不想与佛教界为敌，我不想得罪太多的人：我以后尽量少泄密……
